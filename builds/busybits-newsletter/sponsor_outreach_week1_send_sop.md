# BusyBits Sponsor Outreach — Week 1 Send SOP

## Goal
Book 1-2 sponsor calls this week from Tier-1 health-tech brands.

## Inputs
- Draft copy: `memory/projects/busybits-sponsorship-drafts.md`
- Tracker: `builds/busybits-newsletter/sponsor_outreach_week1_tracker.csv`

## 20-Minute Daily Execution Block
1. Fill `contact_name` + `email` for the day’s rows.
2. Personalize first 2 lines with one real usage detail.
3. Send in EST 09:00-11:00 window.
4. Mark `status` as `SENT` and add timestamp in `notes`.
5. Queue follow-ups in the same row (`followup_1_day`, `followup_2_day`).

## Personalization Tokens
- [Name]
- [Time using product]
- [Specific metric/story]
- [Relevant CTA]

## Success Criteria
- 5/5 Tier-1 sends completed by Thursday.
- Follow-up #1 queued for all 5 by Sunday.
- At least 1 positive reply by next week.
