# BusyBits Newsletter Strategy

- **Subscribers:** ~100k
- **Focus:** High Performance for Busy People.

1.  **Sponsorships:** Inbound-Ready Outbound System (See `memory/projects/busybits-sponsorships.md`).

3.  **Paid Growth:** Running Meta Ads to grow list profitably (Target CPA < $2.00).
