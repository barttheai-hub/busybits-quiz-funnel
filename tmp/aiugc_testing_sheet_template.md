# AI UGC — ad testing sheet template (copy into Sheets/Airtable)

## Tabs
1) **Creative Backlog** (ideas)
2) **Active Tests** (what’s live)
3) **Winners / Learnings**

---

## 1) Creative Backlog (columns)
- Creative ID (auto)
- Offer
- Platform (Meta)
- Angle (skeptical / mistakes / routine / etc.)
- Hook (exact text)
- Mechanism (1 line)
- CTA (exact text)
- Script link (doc)
- Shotlist link (doc)
- Notes / compliance flags
- Status (idea / scripted / produced / ready)

## 2) Active Tests (columns)
- Creative ID
- Ad name
- Launch date
- Audience (broad / interest / LAL)
- Placement (FB/IG/Reels)
- Budget/day
- Spend
- Impressions
- Hook rate (3s views / impressions)
- ThruPlay / 15s views
- CTR (link)
- CPC
- Landing page CVR
- CPA/CAC
- ROAS
- Decision (kill / keep / iterate)
- Next iteration (single variable)

## 3) Winners / Learnings (columns)
- Creative ID
- What won (hook/angle/CTA)
- Why we think it worked
- How to clone it (rules)
- Next 3 variations

---

## Default test matrix (9 ads)
- Hooks: H1 / H2 / H3
- Angles: A1 / A2 / A3
= 9 combinations.

Keep constant:
- Caption style
- CTA destination
- Visual style

Change one variable per iteration.
