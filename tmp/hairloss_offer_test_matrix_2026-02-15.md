# Hair-Loss Funnel: Offer Test Matrix (Rapid CRO)

## Goal
Increase CVR from VSL click to checkout purchase by testing risk-reversal + value stack without changing traffic quality.

## Test 1 — Guarantee Framing
- Control: 30-day money-back
- Variant A: 90-day "Hairline Confidence Guarantee" (with compliance-safe wording)
- Variant B: "Use it 60 days, if not satisfied we'll refund"

## Test 2 — Offer Stack
- Control: Product only
- Variant A: Product + "Doctor Question Prep" PDF
- Variant B: Product + "12-week consistency tracker" + SMS reminder template

## Test 3 — Price Architecture
- Control: Single purchase
- Variant A: Subscribe & save default selected
- Variant B: 2-bottle bundle anchor + per-bottle savings shown

## Test 4 — CTA Copy
- Control: "Buy Now"
- Variant A: "Start My Hairline Plan"
- Variant B: "Lock In My 90-Day Plan"

## Measurement
- Primary: checkout conversion rate
- Secondary: AOV, refund rate, subscription take rate
- Minimum runtime: 1 full buying cycle or 500 checkout sessions/variant

## Next Actions
1. Implement copy-only tests first (guarantee + CTA) for fastest learning.
2. Launch stack test next after one winner emerges.
3. Keep compliance pass before publish (no cure/regrowth certainty claims).
