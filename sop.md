# BusyBody Influencer Outreach SOP

## Tools Needed
- Instagram/TikTok app or web for DMs
- Google Sheets/CSV for tracking (status: Sent, Opened, Replied, Converted)
- Email client if contacts found (use DM primarily)
- Link shortener (bit.ly) for affiliate links
- Canva for custom graphics if needed

## Sequence (per Influencer)
1. **Day 1: Initial Outreach (DM preferred, Email if available)**
   - Use personalized script from outreach-scripts.md
   - Send between 9AM-12PM their TZ (assume US)
   - Track sent date/handle

2. **Day 4: Follow-up 1**
   - 'Hey [Name], circling back on BusyBody collab - any interest?'
   - Attach app store link/screenshot if replied.

3. **Day 8: Follow-up 2 (Value Add)**
   - Share user testimonial or early beta feedback: 'Users loving the quick logs!'
   - Re-offer incentives.

4. **Day 14: Final Touch / Close**
   - If no reply, 'No worries, keeping door open!'
   - Mark as Nurture or Archive.

## Best Practices
- Personalize EVERY message (reference specific content)
- Send 5-10/day to avoid burnout/spam flags
- Respond within 1h to replies
- Goal: 20% response rate → 4 collabs → 1k downloads
- Log responses in CSV: add column 'response_notes'
- Prioritize top 10 by fit_reason first
