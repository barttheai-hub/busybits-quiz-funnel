# Editor / VA Handoff

## Folder handoff format
Each job folder should contain:
- `brief.md`
- `script.json`
- `qc.json`
- `prompts/nanobanana.json`
- `prompts/sora_kling.json`
- `assets_manifest.json`

## VA responsibilities
1. Fill brief from product page and offer notes.
2. Run workflows 01-06.
3. Export with workflow 07.
4. Run compliance check (08) and flag risks.

## Editor responsibilities
1. Assemble timeline per shotlist timestamps.
2. Add captions in safe zone.
3. Use approved CTA overlays only.
4. Ensure pacing and hook clarity in first 3 seconds.
5. Export final variants (A/B hooks, A/B CTA).

## QA pass before launch
- Claim language softeners applied.
- No hard guarantees.
- Audio sync clean.
- Final subtitle pass.
- Human sign-off recorded.
