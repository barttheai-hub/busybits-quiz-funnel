# How to Run (5-Minute Setup)

## 1) Paste product info (60 sec)
In n8n, start with input fields:
- product
- mechanism
- audience
- offer
- claim boundaries

## 2) Generate angle/hooks + script (90 sec)
Run:
- `01_generate_angle_and_hook`
- `02_generate_script`
Pick 1 top hook and one script duration.

## 3) Generate shotlist + prompts + VO (90 sec)
Run:
- `03_generate_shotlist`
- `04_generate_prompts_nanobanana`
- `05_generate_prompts_sora_kling`
- `06_generate_voiceover`

## 4) Export folder + QC (60 sec)
Run:
- `07_export_job_folder`
- `08_qc_and_compliance`

## 5) Generate 10 ads/day cadence
- Batch 5 hooks x 2 tone variants per day.
- Keep one control ad live.
- Kill losers after threshold spend.
- Promote winners into new variations.

Suggested operating split:
- Morning: 5 new ads
- Evening: 5 new ads
- End-of-day: QC + performance log
