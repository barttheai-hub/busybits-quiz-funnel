# CORE PROMPTS (Copy/Paste)

## A) Hook generator (any niche)
Generate 20 UGC ad hooks for this niche/product. Constraints: no guaranteed claims and no policy-violating promises. Use softeners like “may help” or “can support” when needed. Speak like a real person on iPhone. Make hooks pattern-based: shock, confession, myth-bust, proof, routine. Return as JSON with: hook, angle, why_it_works, risk_words_to_avoid.

## B) Script generator (45s UGC)
Write a 45s UGC ad script:
0–3s hook
3–10s credibility
10–25s problem + mechanism
25–40s proof/routine
40–45s CTA
Output: spoken lines + on-screen text + b-roll notes.
Tone: blunt, fast, ‘friend giving advice’.

## C) NanoBanana keyframe prompt template
Photoreal iPhone UGC frame, 9:16, natural lighting, handheld feel.
Scene: {scene_description}
Subject: {avatar_or_actor_description}
Props: {props}
On-screen text: {text_overlay} (minimal, 3–6 words)
Style: raw UGC, not polished commercial.
Negative: blur, extra fingers, watermarks, UI overlays, distorted face.

## D) Kling/Sora motion prompt template
Animate the keyframe into a 3–5s clip.
Motion: {camera_motion}, {subject_action}
Keep identity consistent, no face warping.
Style: handheld iPhone, slight shake, natural movement.
Avoid: glitch, morphing, text jitter, random object changes.

## E) ElevenLabs VO direction template
Voice: confident, conversational, slightly urgent.
Pacing: fast but clear.
Emphasis: underline the hook and mechanism lines.
Add 1–2 natural pauses for impact.
