# 02 — Direct Response Script Template (45s)

## Structure

### 0:00–0:03 Hook
- **Visual:** [Pattern interrupt visual]
- **VO:** [Audio hook from research step]

### 0:03–0:12 Agitation
- **VO Template:**
  "If you’ve tried [status quo #1] and [status quo #2] but still struggle with [pain], you’re not the problem. Those only treat [symptom], not [root cause]."

### 0:12–0:25 Mechanism Pivot
- **VO Template:**
  "[Product] works differently. It uses [mechanism] to target [root cause] so you can finally [desired outcome]."

### 0:25–0:35 Proof/Demo
- **VO Template:**
  "Here’s what that looks like in practice: [specific demo/proof moment]."

### 0:35–0:45 CTA + Risk Reversal
- **VO Template:**
  "If you want [outcome], tap ‘Shop Now’ and try [product] with [guarantee/risk-reversal]."

## Variants to test
- **CTA A (Outcome):** "Get started today."
- **CTA B (Urgency):** "First batch pricing ends soon."
- **CTA C (Risk-free):** "Try it, and if it’s not for you, you’re covered."
