# Quickstart Checklist (20 Minutes)

## 0–5 min
- Pick one product + one audience.
- Run the Research Agent prompt.
- Select top 2 hooks.

## 5–10 min
- Build script using the DR template.
- Create 4 scene prompts.

## 10–15 min
- Generate visuals (or storyboard frames).
- Add VO + captions.

## 15–20 min
- Export first draft ad.
- Launch as test creative against a control.

## First test metrics to track
- Thumbstop rate
- 3-second plays
- CTR
- CPA
