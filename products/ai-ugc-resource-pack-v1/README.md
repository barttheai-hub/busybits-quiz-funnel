# AI UGC Resource Pack (v1)

Thanks for purchasing.

This pack helps you go from **offer URL -> hook -> script -> scene prompts -> production-ready ad concept** quickly.

## What’s inside
1. `01-research-agent-prompt.md` — prompt to generate viral hooks from market inputs.
2. `02-direct-response-script-template.md` — 45-second ad script framework.
3. `03-visual-consistency-guide.md` — keep the same AI actor across scenes.
4. `04-n8n-workflow-blueprint.md` — points to the working n8n workflow JSON.
5. `workflows/01_ai_ugc_webhook_pipeline.json` — importable, runnable n8n workflow.
6. `workflows/01_ai_ugc_webhook_pipeline.runbook.md` — setup + test payload.
7. `05-hairloss-demo-assets.md` — complete worked example.
8. `quickstart-checklist.md` — 20-minute setup path.

## Suggested workflow (fast)
- Run the Research Agent prompt with your product URL.
- Pick 1 winning hook and build your script with the template.
- Convert script into scene prompts using the consistency guide.
- Use the n8n blueprint to automate for each new product/audience.
